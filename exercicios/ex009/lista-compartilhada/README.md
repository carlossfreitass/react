# Lista Compartilhada
Data: 02/11/2025

Tema: JSX, Props, CSS Modules, React Fragments, Eventos, UseState, Renderização Condicional, Renderição de Listas e State Lift

Author: Carlos Freitas

## Descrição
Este exercício tem como objetivo colocar em prática o uso do State Lift no React.

## Objetivo
Criar uma aplicação em React que permite adicionar itens a uma lista e exibi-los dinamicamente na tela, além de ser possível limpar todos os elementos com um botão dedicado.

## Como executar
Para rodar este exercício localmente:

`npm install`

`npm start`
