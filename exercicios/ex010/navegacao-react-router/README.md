# Navegação com React Router
Data: 03/11/2025

Tema: JSX, CSS Modules, React Router

Author: Carlos Freitas

## Descrição
Este exercício tem como objetivo colocar em prática o uso do React Router.

## Objetivo
Desenvolver uma aplicação em React com navegação entre múltiplas páginas utilizando o React Router DOM.

## Como executar
Para rodar este exercício localmente:

`npm install`

`npm start`
